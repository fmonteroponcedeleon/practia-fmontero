Practia - Prueba Técnica

Esta prueba fue desarrollada por el Lic. Francisco Montero con Angular CLI 8.3.8 y Node 8.9.4. Se utilizaron paquetes npm como por ejemplo Bootstrap 4.4.1, JQuery 3.4.1, Angular Font Awesome 3.1.2, Ng2 Search Filter 0.5.1, Ngx Order Pipe 2.0.4, Ngx Toastr 11.3.3, entre otros.

  • Generar una aplicación en Angular 8

  • Generar listado de Servicios
  • Generar Alta, Baja y Modificaciones de Servicios

  • Permitir desvincular un servicio a un cliente
    • Consultar al usuario si esta seguro que desea concretar la operación
    • Diseñarlo de tal manera que pueda ser reutilizable

  • Validar en todos los formularios que ningún campo quede vacío
